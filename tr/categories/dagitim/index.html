IIS üzerinde Node.js Dağıtım ve Yayınlama - iisnode

IIS üzerinde Node.js Dağıtım ve Yayınlama - iisnode Windows Server 2016 x64 ve IIS 10.0 üzerinde test edilmiştir. Farklı sürümler için tüm gereksinimler aynı sadece iisnode için uygun sürüm kurulmalıdır. Gereksinimler: WebPlatformInstaller: WebPlatformInstaller IIS URL Rewrite extension: IIS URL Rewrite extension Node.js: node.js iisnode v0.22.1 x64 : iisnode v0.22.1 x64 Kurulum sonrasında projede “web.config” bulunmalıdır. Aşağıdaki config örneği Express Framework ile de kullanılabilir. App, server.js ile başlatılacaktır. Konsoldaki çıktılar logDirectory ile belirtilen klasorden kontrol edilebilir.
