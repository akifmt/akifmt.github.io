Raspberry Pi Remote Desktop Turkish Keyboard Problem

Problem: Raspbery Pi Remote Connection TR Keyboard Not Recognizing Solution: missing km-041f.ini file Copy the km-041f.ini file to the /etc/xrdp/ directory on the Raspberry Pi. When xrdp is restarted, TR characters are no longer a problem. The thing to note is that you must access the directory as root.
